# MERN MySQL Auth CRUD
 Description

A full-stack web application with user authentication and dashboard CRUD operations using React, Node.js, Express, and MySQL.

Tech Stack

* React.js
* Node.js
* Express.js
* MySQL

 Setup

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

## Database

Import `database.sql` into MySQL.

##  Features

* User Registration & Login
* JWT Authentication
* CRUD Operations
